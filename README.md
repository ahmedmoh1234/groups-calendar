## Groups Calendar

This app was created using https://getmocha.com.
Need help or want to join the community? Join our [Discord](https://discord.gg/shDEGBSe2d).

To run the devserver:
```
npm install
npm run dev
```

To run the dev server on your local network:
```
npm run dev -- --host
```

Use `npm run dev -- --host` when you want to access the app from another device on the same network.

Local development runs at `/`.
Production builds use `/groups-calendar/` via `.env.production`, which keeps GitHub Pages deployments working.
